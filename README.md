# TRACKBACK (CAREER GUIDANCE WEBSITE)

## Features
- **Career Exploration**: Kids can explore career options in Science, Commerce, and Arts.
- **Field-Specific Questions**: Each field has a unique set of questions to guide users.
- **Interactive Quiz**: A final quiz assesses user interests and skills to suggest the best career options.
- **Trending Opportunities**: Kids can explore the trending job opportunities and watch videos explaining various fields in depth.
- **Personalized Suggestions**: A test button leads to a form that collects user details for better recommendations.
- **AI-Powered ChatBot**: Kids can interact with AI to resolve doubts or queries for better clarification.
- **Mentor Guidance**: Students can receive mentorship and career advice from industry experts.
- **Scholarships**: Information on government scholarships to help parents understand financial expenditures.
- **Bilingual Support**: The website is available in English and Hindi for better accessibility.

## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/your-repo/trackback.git
   ```
2. Navigate to the project directory:
   ```sh
   cd career-guidance
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm start
   ```

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express.js
- **AI & Data Processing**: Python
- **Database**: JSON (for career data storage and retrieval)

## Usage
1. Select a career field (Science, Commerce, or Arts).
2. Answer field-specific questions.
3. Take the final quiz to determine career suitability.
4. Fill out the form for personalized recommendations.
5. Interact with the AI Chatbot to resolve queries.
6. Explore trending job opportunities and professions.
7. Receive mentorship and career advice.
8. Check for available scholarships.

## AI-Powered Career Guide for Students in Rural India
### **Scenario:**
Ravi, a 16-year-old student from a small town in Bihar, is passionate about technology but has no exposure to career counseling. His school lacks resources, and his parents want him to take up farming like previous generations. With no guidance on emerging job markets, scholarships, or skill development, Ravi feels lost about his future.

### **Solution:**
- **AI-Powered Chatbot & Mentor Guidance:** Ravi can interact with the AI chatbot for career advice and access mentorship programs from experts.
- **Trending Jobs & Professions:** The platform showcases emerging career options in technology and other fields, helping Ravi explore new opportunities.
- **Personalized Career Pathways:** Based on Ravi’s interests and responses, the AI system recommends suitable careers and study plans.
- **Scholarships & Financial Aid:** The platform provides information on government scholarships to help Ravi’s parents understand financial assistance options.
- **Multilingual Support:** The website is available in Hindi and English, ensuring ease of use for Ravi and his family.
